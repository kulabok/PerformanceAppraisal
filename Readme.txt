Performance Appraisal project.

This program checks all incoming requests for existing record in DB and allows access to a personal cabinet
considering his position level in company.

Ceo can:
see all tasks for all employees which have expired period;
see all his tasks;
add new superiors;
add new tasks for all levels of employees;
add progress to any task;
setup minimal performance rates to achieve bonuses for all levels of tasks;

Superior can:
see all his/her tasks;
add new subordinates;
add new tasks for all employees of department he/she manages;
add progress to any task for any employee of the department he/she manages;

Employee can:
see all his/her tasks;
add progress to any his/her task;

All data stores in MySql database.
